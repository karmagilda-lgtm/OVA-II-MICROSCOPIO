OVA Microscopio óptico

Procedimiento para enfocar

Introducción al procedimiento

“Bienvenido al laboratorio virtual. En este OVA aprenderás a usar correctamente un microscopio. Seguirás pasos interactivos que te permitirán observar cualquier muestra con precisión y seguridad.”

¡Comencemos!

 

Paso 1 . Colocar la Muestra



“Selecciona la muestra que deseas observar y colócala sobre la platina del microscopio.”


Paso 2. Coloque la muestra en la platina


Paso 3. Enfoque con objetivo 10x

  1. Levanta la platina hasta su límite superior.
  2. Gira el revolver para colocar el objetivo de 10x.
  3. Ajusta lentamente el enfoque con el tornillo macrométrico hasta que la muestra comience a aparecer.
  4. Enfoca con precisión usando el tornillo micrométrico para observar los detalles.



4. Lograras ver una imagen como esta una ves que suabemente ajuste con el tornillo micrométrico

Paso 4. Enfoque con objetivo 40x

  1. Baja ligeramente la platina para evitar que el objetivo toque la muestra.
  2. Gira el revolver hasta colocar el objetivo de 40x en posición.
  3. Ajusta con cuidado el enfoque usando el tornillo macrométrico, solo un poco, hasta que la imagen comience a verse.
  4. Afina los detalles con el tornillo micrométrico, hasta obtener una imagen nítida y clara.


     





Observa con atención los detalles de la muestra a este aumento intermedio.


   


Paso 5 Enfoque con objetivo seco fuerte (100X)

Procedimiento con el objetivo seco fuerte (100x)

  1. Baja la platina cuidadosamente para evitar dañar la muestra o el objetivo.
  2. Gira el revolver hasta colocar el objetivo de 100x en posición.
  3. Coloca una gota de aceite de inmersión sobre la muestra (si se trata de un objetivo de inmersión).
  4. Eleva lentamente la platina hasta que el aceite y el objetivo entren en contacto, sin ejercer presión.
  5. Enfoca con cuidado utilizando únicamente el tornillo micrométrico, hasta obtener una imagen clara y bien definida.
  6. Observa con precisión los detalles más finos de la muestra, evitando movimientos bruscos del enfoque.
